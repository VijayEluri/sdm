Vigtigt!

XML-skemaerne i denne folder er lokale XML-skemaer, der indeholder og validerer præcis
den samme OIOXML-struktur som de tilsvarende OIOXML-skemaer på Digitaliser.dk.
Disse lokale XML-skemaer er organiseret i langt færre fysiske XML-filer end de
normative OIOXML-skemaer samt refererer ikke til OIOXML-skemaer på Digitaliser.dk.
Derfor egner denne skemaform sig bedre til validering i produktionsmiljøer, heraf
navnet produktionsskemaer.
